# SSH Project

Learning about SSH (Secure Shell) protocol and how to securely connect to remote servers.

## Tasks

- **0-use_a_private_key**: Bash script to connect to server using SSH private key
- **1-create_ssh_key_pair**: Bash script to create RSA key pair
- **2-ssh_config**: SSH client configuration file

## Requirements

- All scripts must be executable
- All scripts start with `#!/usr/bin/env bash`
- All scripts have a comment explaining what they do
