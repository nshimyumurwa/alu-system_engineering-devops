# Web Server Configuration Project

This project contains bash scripts for configuring and managing an Nginx web server on Ubuntu 16.04 LTS.

## Tasks

### 0. Transfer a file to your server
Script that transfers a file from client to server using scp.

### 1. Install nginx web server
Script that installs and configures Nginx to serve a page containing "Holberton School".

### 2. Setup a domain name
Domain name configuration for the web server.

### 3. Redirection
Script that configures Nginx with a 301 redirect from /redirect_me.

### 4. Not found page 404
Script that configures a custom 404 error page.

### 5. Design a beautiful 404 page
Custom HTML design for the 404 error page.

## Requirements
- Ubuntu 16.04 LTS
- Nginx web server
- All scripts must be executable
- Scripts must pass Shellcheck (version 0.3.7)
